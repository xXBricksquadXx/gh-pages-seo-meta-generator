Static (no-build) web tool to generate SEO-friendly <head> tags for GitHub Pages:

- Title + description + canonical
- Open Graph + Twitter cards
- JSON-LD (WebSite + WebPage)
- One-click copy
- Shareable links (form state stored in the URL hash using base64url)

## Run locally

```powershell
python -m http.server 5173
# then visit http://localhost:5173
```
